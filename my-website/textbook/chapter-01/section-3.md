# 1.3 Core Concepts

By the end of this textbook, you will be able to understand:

## ⭐ Core Concepts
- What makes a robot "autonomous"
- How sensors gather information
- How AI models interpret the environment

## ⭐ Building Blocks
- Camera vision, image processing, object detection
- Motion planning & navigation
- Robot control mechanisms (PID, kinematics)

## ⭐ Practical Skills
- Building small physical AI projects
- Writing Python code for robotics
- Integrating sensors like ultrasonic, IMU, camera
- Controlling motors, servos, actuators
- Running AI on microcontrollers or mini-computers (Arduino, ESP32, Raspberry Pi)

Each chapter introduces theory + hands-on experiments so you learn by *doing*.
