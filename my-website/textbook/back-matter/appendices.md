# Appendices

## Appendix A: Hardware Specifications
This appendix provides detailed specifications for common hardware components used in Physical AI systems.

### Microcontrollers
- **Arduino Uno**: ATmega328P, 16MHz, 32KB flash, 2KB SRAM
- **Raspberry Pi 4**: Broadcom BCM2711, 1.5GHz, 1GB-8GB RAM
- **ESP32**: Dual-core Xtensa LX6, 240MHz, 4MB flash, WiFi/Bluetooth

### Sensors
- **IMU (MPU6050)**: 3-axis gyroscope, 3-axis accelerometer, I2C interface
- **Camera (Pi Camera)**: 8MP, 1080p video, CSI interface
- **Ultrasonic (HC-SR04)**: 2cm-400cm range, 4mm accuracy

### Actuators
- **Servo (SG90)**: 180° rotation, 9g weight, PWM control
- **DC Motor (TT Motor)**: 3-6V, 200 RPM, with encoder option
